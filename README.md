# jetic_bf
## Supported functions:
* Variables:
> from 'a' to 'z', and system variable '_'.
* '+' and '-':
> '+x' will be '+' repeat x times. '1' in '+1' cannot be omitted.
* ' ':
> space is ignored, so feel free to add ' ' in jetic_bf code to make it easier to read.
* '~':
> '~x' will clear the value of x to 0.
* '=':
> '=ab' means b=a
